# 📈 A股策略回测分析工作台

基于 FastAPI + ECharts 的本地化 A股多策略回测系统，支持实时进度展示与绩效报告生成。

## ✨ 功能特性

- **多策略支持**：RSV超卖、5日强平动量、超跌反弹三套策略
- **实时SSE推送**：回测过程中实时展示进度与交易流水，无需等待全部完成
- **可视化报告**：收益率走势图、最大回撤、胜率、盈亏比等完整绩效指标
- **动态参数配置**：持仓上限、初始资金、回测周期等均可在首页自由配置
- **交易流水分页**：支持按页浏览全部历史交易记录

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install fastapi uvicorn pandas numpy akshare tqdm
```

### 2. 更新/下载股票数据（重要！）

> 由于股票历史数据体积较大，不纳入代码仓库。建议使用腾讯行情接口进行快速增量更新：

```bash
# 每日增量更新（多线程秒级同步）
python append_tencent.py
```

该脚本会自动：
- 自动识别当前市场内所有 A股文件
- 使用腾讯接口多线程并发拉取最新数据（含前收盘价、涨跌幅、换手率等核心字段）
- 自动补充静态字段与自愈缺失数据

脚本会自动：
- 获取全市场 A股股票列表（沪深两市）
- 多线程并发下载近一年的日线数据
- 支持断点续传，中断后可重新运行继续下载
- 数据保存至 `all_stock_data/` 目录

> ⚠️ 首次下载约需 **5~15 分钟**，请耐心等待。

### 3. 启动服务

```bash
python -m uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

### 4. 打开浏览器

访问 [http://127.0.0.1:8000](http://127.0.0.1:8000) 即可开始使用。

## 🧩 支持的策略

| 策略名称 | 选股逻辑 | 买入时机 | 卖出条件 |
|---------|---------|---------|---------|
| RSV超卖 | 长RSV≥80 且 短RSV≤20，成交量缩量排名 | T+1日开盘 | 持仓满2天收盘价卖出 |
| 5日强平动量 | MA5穿越MA20金叉 | T+1日收盘价 | 盈利或持仓满5天 |
| 超跌反弹 | KDJ深水超卖 + MA60趋势 + 砖型红绿柱反转 | T+1日开盘（跳空≤0.5%过滤） | 当日盈利卖出或持仓满5天强制清仓 |

## 🛠️ 技术栈

- **后端**：Python 3.10+ / FastAPI / Uvicorn
- **数据获取**：AKShare
- **前端**：HTML + Tailwind CSS + ECharts
- **实时推送**：Server-Sent Events (SSE)
